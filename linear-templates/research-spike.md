# Research / Spike

## Recommended form fields

- `Title` (property field, required)
- `What question needs answering?` (long text, required)
- `Why does this matter?` (long text, required)
- `Area` (dropdown: product, frontend, backend, infra, design, AI)
- `Expected output` (dropdown: recommendation, prototype, comparison, decision doc)
- `Reference links` (long text, optional)

## Recommended default properties

- Status: `Triage`
- Label: `research`
- Team: preset if this template belongs to one team only

## Description body

```md
## Research question
State the decision or uncertainty clearly.

## Context
Summarize the current situation and why this needs investigation.

## Deliverable
Specify the expected output: recommendation, prototype, tradeoff memo, or benchmark.

## Boundaries
List what is explicitly out of scope.
```
