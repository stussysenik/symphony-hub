# Bug Report

## Recommended form fields

- `Title` (property field, required)
- `What happened?` (long text, required)
- `What did you expect?` (long text, required)
- `Where is it happening?` (dropdown: frontend, backend, infra, mobile, design)
- `Severity` (dropdown: blocker, high, medium, low)
- `Screenshots / recordings / links` (long text, optional)

## Recommended default properties

- Status: `Triage`
- Label: `bug`
- Team: preset if this template belongs to one team only

## Description body

```md
## Summary
Describe the bug in one or two sentences.

## Actual behavior
What went wrong?

## Expected behavior
What should happen instead?

## Scope / surface
Which app, page, workflow, or API is affected?

## Evidence
Attach screenshots, recordings, logs, or URLs if available.
```
